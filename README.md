# 🧠 NTSGROW - The Ultimate Brain Training Hub

**NTSGROW** is a high-performance web platform designed to enhance cognitive functions through interactive, wordless gaming. Built with a "Classic-Modern" aesthetic, it focuses on sharpening memory, logic, reaction time, and spatial awareness.

## 🚀 Live Website
Experience the games here: 
👉 [https://inox9103.github.io/ntsgrowgames/index.html](https://inox9103.github.io/xprimegames/index.html)

## 🎮 The Training Suite (10 Games)

| Game | Cognitive Focus | Description |
|:--- |:--- |:--- |
| **Memory Matrix** | Visual Memory | Recall complex tile patterns. |
| **Logic Flow** | Spatial Reasoning | Solve the classic 15-puzzle grid. |

| **Pattern Master** | Focus & Attention | Memorize and repeat light/sound sequences. |

## ✨ Key Features
* **Minimalist Design:** Dark-themed UI for reduced eye strain and high focus.
* **Fully Responsive:** Seamlessly playable on Desktops, Tablets, and Smartphones.
* **Immersive Audio:** Each game features custom background music for a deeper experience.
* **Performance Scaling:** Games get progressively harder as your brain speed increases.
* **Lightweight:** Built with pure HTML5, CSS3, and Vanilla JavaScript (No heavy libraries).

---

## 🛠️ Technology Stack
- **Frontend:** HTML5, CSS3 (Flexbox & Grid)
- **Logic:** Vanilla JavaScript (ES6+)
- **Audio:** HTML5 Audio API
- **Hosting:** GitHub Pages
